Extract data for the RTB toolchain

Required layers an their content:

traeger
Beam points marked with text entries, containing the the appropriate beam point code (see below)
beton
Contour points of the beton as point entries. The order of the points has to be continous.
podest
Podest levels marked with points
diverse
Lowest and highest concrete pressure point as point entry, number of beams as text as "T=60"

Beam point codes:

0
Pure geometry point
1
Flexible joint with butt plates
2
Compression member
3
Tension member M30
4
Tension member M20
5
Form-giving ring rectangular hollow 80/80/5
6
Form-giving ring rectangular hollow 100/100/5
8
Rigid connection with butt plates
9
Rigid connection with 3 axial screws

Extract truss data

Use your favorite drafting software to visulally prepare the data. Create a layer named nodes. On this layer, mark the position of the nodes with a simple text entry. The insertion point of the text has to be identical with exact location of the node. Draw the middle axes of the rods on layer rods. The endpoints of each rod have to fit to one of the nodes. That means even long straight rods have to be interrupted at nodes.
